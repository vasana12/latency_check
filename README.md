# latency_check RESt API
- rest api 요청 latency 측정
- 요구사항
  1. 임의의 application 의 url 호출시 latency 데이터를 전송받아 csv 형식으로 저장
  2. 임의의 application cpu(%), memory(Bytes) 사용량 전송받아 이를 저장
  3. 위 1,2 데이터는 csv 형식으로 조회 할 수 있어야 함.
